# 🍺 BrewCost

## Trabajo Práctico Integrador – Arquitectura y Sistemas Operativos

Aplicación desarrollada en **Python** como parte del Trabajo Práctico Integrador de la materia **Arquitectura y Sistemas Operativos** de la **Tecnicatura Universitaria en Programación a Distancia – Universidad Tecnológica Nacional (UTN)**.

El proyecto consiste en la implementación de un entorno virtualizado utilizando **Oracle VirtualBox**, la instalación de **Linux Mint 22.3 XFCE** y el desarrollo de una aplicación denominada **BrewCost**, destinada al cálculo del costo de producción de un lote de cerveza artesanal.

---

## Objetivos

* Implementar una máquina virtual mediante Oracle VirtualBox.
* Instalar y configurar Linux Mint 22.3 XFCE.
* Instalar las VirtualBox Guest Additions.
* Desarrollar una aplicación en Python dentro del entorno virtualizado.
* Demostrar el funcionamiento de un sistema operativo invitado ejecutando una aplicación real.

---

## Tecnologías utilizadas

* Python 3
* Linux Mint 22.3 XFCE
* Oracle VirtualBox 7
* Windows 10

---

## Funcionalidades

La aplicación permite:

* Ingresar el nombre del lote.
* Ingresar la cantidad de litros producidos.
* Registrar el costo de la malta.
* Registrar el costo del lúpulo.
* Registrar el costo de la levadura.
* Calcular el costo total del lote.
* Calcular el costo por litro.
* Identificar el ingrediente de mayor costo.
* Identificar el ingrediente de menor costo.

---

## Ejecución

Desde la terminal:

```bash
python3 calculadora_costos_cerveza.py
```
